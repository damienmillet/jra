Documentation de l'API Utilisateurs

  • GET /api/users
  • POST /api/users
  • GET /api/users/{id}
  • PUT /api/users/{id}
  • DELETE /api/users/{id}

Endpoints de l'API

GET /api/users

Récupère la liste de tous les utilisateurs.

GET /api/users HTTP/1.1
Host: example.com

POST /api/users

Crée un nouvel utilisateur.

POST /api/users HTTP/1.1
Host: example.com
Content-Type: application/json

{
    "name": "John Doe",
    "email": "john.doe@example.com"
}

GET /api/users/{id}

Récupère les détails d'un utilisateur spécifique par son ID.

GET /api/users/{id} HTTP/1.1
Host: example.com

PUT /api/users/{id}

Met à jour les informations d'un utilisateur spécifique par son ID.

PUT /api/users/{id} HTTP/1.1
Host: example.com
Content-Type: application/json

{
    "name": "Jane Doe",
    "email": "jane.doe@example.com"
}

DELETE /api/users/{id}

Supprime un utilisateur spécifique par son ID.

DELETE /api/users/{id} HTTP/1.1
Host: example.com
